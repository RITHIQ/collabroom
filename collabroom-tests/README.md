# Collabroom Test Suite

| Command | Description |
|---------|-------------|
| `npm run test:web` | Selenium web E2E |
| `npm run test:mobile` | Appium mobile E2E |
| `npm run test:dast` | Security scan |
| `npm run test:all` | All suites + HTML report |

Requires web app on **http://localhost:3000** for local runs.

Demo credentials: see [docs/TESTING.md](../docs/TESTING.md).
