## 🐛 Troubleshooting

### Data Not Loading
```
Hard refresh: Ctrl+Shift+R (Chrome) or Cmd+Shift+R (Mac)
Check: localStorage has 50MB per domain
Clear: localStorage in DevTools → Application tab
```

### Contracts Not Visible
```
Make sure logged in as creator account
Check: Switch to different role if needed
Wait: 2-3 seconds for data loading
```

### Payment Modal Stuck
```
Close and reopen
Check: Browser console for errors (F12)
Try: Different browser (Chrome recommended)
```

### PDF Download Failed
```
Check: Browser popup settings
Try: Download to different location
Use: Chrome or Firefox (best PDF support)
```

### Signature Canvas Not Responding
```
Click inside canvas area first
Try: Different mouse/touchpad
Clear: Canvas buffer and retry
Use: Mouse over touchpad
```
