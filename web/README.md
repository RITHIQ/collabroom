# Collabroom Web App

React + Vite frontend. See the [root README](../README.md) and [docs/](../docs/) for setup.

```bash
npm install
npm run dev      # http://localhost:3000
npm run build
npm run preview
```

Environment: copy `.env.example` to `.env.local` and set `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY`.
